Website Personal Profile

Sebelumnya, website ini akan berisi informasi personal mengenal diri saya dan sebagai upaya untuk menyelesaikan submission saya di kelas Dasar Pemrogramman Web di platform Dicoding Indonesia yang saya ambil.

Kenalan Dulu Yuk!

Perkenalkan nama saya Sahrul Milad, saya lahir di kota Indramayu pada tanggal 16 Juni 1998. Saya menempuh pendidikan di SMK Negeri 1 Losarang degan mengambil prodi Teknik Elektronika Industri tahun ajaran 2014 - 2017. Saya sendari awal memang lebih menyukai hal-hal seputar Technology dan Engineering. dan selama SMK saya aktif dalam kegiatan seputar keilmuan Elektronika Industri, seperti di Himpunan Siswa Elektro (HISEL) dan Robotik Riloz. dan Alkhamdulillah juga saya berkesempatan mengikuti Lomba Kompetensi Siswa tingkat Kabupaten hingga tingkat Provinsi, dengan mengambil Mata Bidang Lomba Elektronika Aplikasi. Kemudian saya melanjutkan study di Politeknik Negeri Indramayu dengan mengambil prodi Technic Refrigeration and Aircoditioning. Selama proses pendidikan pun saya aktif di keorganisasian Forum Mahasiswa Bidikmisi (FORMADIKSI POLINDRA) dan Robotika Polindra. Dan Alkhamdulillah bisa menyelesaikan study di tahun 2021. Dan sekarang saya ingin mencoba hal baru khususnya di dunia Teknologi. Maka saya mulai mempelajari keilmuan seputar dunia Teknologi, diawali dengan mempelari seputar dunia Web Engineer / Web Developer secara otodidak melalui Youtube, Sosial Media, Google hingga platform-platrom e-course IT seperti dicoding ini. Inilah saya, Salam Kenal buat semuanya. Mohon bantuannya.
